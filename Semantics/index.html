Why use semantics?

  • accessability
  • searchability
  • internationalization
  • interoperability

How to apply this on HTML?

  • Use HTML-tags correctly.

  • Separation of
    content and presentation.    Wireframe. Slice. Prepare first then code.

  • Use correct document outline. Think of your HTML as a book, that has a logical order in it.

  • Write content first.Then style and change layout with CSS.

Where to use a nav element?

  • Header
  • Footer or just above the footer
  • Aside for less important links (ex. recently visited)
  • Table of contents
  • Pagination and next/previous buttons
  • Search forms
  • Breadcrumbs

When to use hr-tag?

The hr-tag represents a thematic break. So you can use is in between sections.

However, since the seperator is mostly for styling-purposes only, it doesn't harm to use a div. Using a div also gives you much more styling options.
